## Todo App built with React Suspense & Hooks

Playing with new toys. :sunglasses:

## Development

`$ npm start`

## Build

`$ npm run build`

To serve it locally, remove "homepage" from `package.json`, run `npm run build`, then `npx serve -s build`.

## Deploy

`$ npm run deploy`
